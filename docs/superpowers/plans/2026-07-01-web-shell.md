# Web Shell (Plan 4a-1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the `app/web/` Next.js app as a themed, bilingual, runnable shell: Reveal-Glow theme (Tailwind + shadcn/ui + Poppins), next-intl locale routing (en/de), an app layout (header with the logo, footer with the "unofficial fan project" disclaimer), and a localized home page.

**Architecture:** A Next.js App Router project in the existing `app/` npm workspace. Scaffolded with the official `create-next-app` and `shadcn` CLIs (version-safe), then themed with Reveal-Glow CSS variables + lesson-color tokens, localized with next-intl path routing (`/[locale]/…`), and wrapped in a header/footer layout. No data yet — search and detail come in 4a-2/4a-3.

**Tech Stack:** Next.js (App Router, latest), TypeScript, Tailwind CSS, shadcn/ui, `next-intl`, Poppins via `next/font`, Vitest + @testing-library/react (component tests), Playwright (e2e).

## Global Constraints

- Node **20+**, TypeScript, ESM. New code under `app/web/`.
- **Dark-first Reveal-Glow** palette (from `logos/BRAND-GUIDE.md`): midnight `#13122A`, ink `#1C1838`, badge `#181634`; gold `#E8B23A` / gold-light `#F6D58B` / gold-dark `#C8881E`; indigo `#3B3194` / `#6E66C9`; parchment `#FBF3DC`.
- **Lesson colors** (facet tokens): `care_of_magical_creatures #5CB878`, `charms #5B8DEF`, `potions #A06CD5`, `transfiguration #E0555B`, `quidditch #EA7B3C`.
- **Poppins** is the house font; SemiBold (600) for headings/wordmark.
- **i18n:** next-intl, path-based `/[locale]/…`, locales `en` (default) + `de`.
- Config env-driven — no hardcoded hosts. This slice needs no backend env yet (no data).
- English identifiers/comments; Conventional Commits.
- The **disclaimer** text (exact): "Revelio is an unofficial fan project. Harry Potter and the card artwork/trademarks belong to Warner Bros.; the TCG to Wizards of the Coast. Not affiliated with or endorsed by them." (translate for `de`).

import { getRequestConfig } from 'next-intl/server'
import { routing } from './routing'

export default getRequestConfig(async ({ requestLocale }) => {
  let locale = await requestLocale
  if (!locale || !routing.locales.includes(locale as (typeof routing.locales)[number])) {
    locale = routing.defaultLocale
  }
  return {
    locale,
    messages: (await import(`../messages/${locale}.json`)).default,
  }
})

- [ ] **Step 5: Move the home page under `[locale]` + locale layout**

Delete the scaffolded `app/web/src/app/page.tsx` (root page). Create `app/web/src/app/[locale]/layout.tsx`:
```tsx
import { NextIntlClientProvider, hasLocale } from 'next-intl'
import { notFound } from 'next/navigation'
import { getMessages } from 'next-intl/server'
import { routing } from '@/../i18n/routing'

export function generateStaticParams() {
  return routing.locales.map((locale) => ({ locale }))
}

export default async function LocaleLayout({
  children,
  params,
}: {
  children: React.ReactNode
  params: Promise<{ locale: string }>
}) {
  const { locale } = await params
  if (!hasLocale(routing.locales, locale)) notFound()
  const messages = await getMessages()
  return (
    <NextIntlClientProvider locale={locale} messages={messages}>
      {children}
    </NextIntlClientProvider>
  )
}
```
Create `app/web/src/app/[locale]/page.tsx`:
```tsx
import { useTranslations } from 'next-intl'

export default function Home() {

      <p className="mt-4 text-lg text-muted-foreground">{t('tagline')}</p>
    </main>
  )
}
```
Note: the import `@/../i18n/routing` resolves `i18n/` at the web root; if the alias doesn't reach it, use a relative import (`../../../../i18n/routing`) — verify the path resolves.
